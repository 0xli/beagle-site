// 关于

一个身份连接通信、智能体与 私有服务 。

Beagle 基于一个简单理念：属于你的身份应该同时适用于人与人通信、智能体协作、文件交换和私有机器访问，而不是让每项能力都变成新的平台账户。

// decent network
Beagle 是产品界面，AgentNet 是机器引擎，Decent Network 是协议。

三个名字描述同一系统的不同层级：Beagle 面向人，AgentNet 面向机器与智能体，Decent Network 提供身份、关系、发现和加密传输。阅读论述与白皮书深入了解。

Beagle 到底是什么

Beagle 是私密通信工具、智能体通信界面，也是进入一张由你拥有的网络的入口。

2026 年 7 月

一个系统，三个名字

Beagle 是人直接使用的产品：手机应用、桌面控制台、会话、联系人、通话与文件。AgentNet 是机器侧引擎：身份、持久化、CLI 自动化、私有虚拟网络、路由与代理。Decent Network 是下面的开放身份、关系、发现与加密传输层。

层负责什么谁使用
Beagle消息、联系人、文件、通话、桌面控制个人与团队
AgentNet智能体收件箱、CLI/TUI、私有局域网、访问与路由智能体、设备、运营者
Decent Network身份、关系、发现与传输应用与基础设施

它改变了什么

  • 身份是本地生成的密钥对，而不是向平台租来的账户。
  • 关系代表明确同意：陌生身份不会天然获得访问机器的能力。
  • 通信可以直连、中继或离线排队，不需要公开 webhook。
  • 同一段关系可以承载消息、文件、智能体交接，或明确授权的私有服务访问。

选择最小且有用的入口

01
在手机上聊天

安装 Beagle App。开始通信不需要安装 AgentNet。

02
连接机器或传文件

在 macOS、Windows/WSL2 或 Linux 上安装 Beagle Desktop + AgentNet。

03
给 AI 智能体一个收件箱

在智能体机器上使用 OpenClaw Beagle Channel 或 AgentNet CLI。

04
构建产品

先用 @decentnetwork/peer 获得身份与消息；已有服务需要私有地址时再加入 @decentnetwork/lan。

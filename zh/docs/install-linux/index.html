安装

在 Linux 上安装

在工作站、服务器或智能体主机上安装持久运行的 AgentNet 节点与本地控制台。

2026 年 7 月

安装并初始化

sudo npm install -g pm2 @decentnetwork/lan
agentnet init
sudo agentnet service install
身份归属

请使用真正会运行 AgentNet 的账户初始化。sudo 只用于安装软件包与服务。

运行本地服务

pm2 start "agentnet proxy router --listen 127.0.0.1:8889" --name agentnet-router
pm2 start "agentnet ui --listen 127.0.0.1:8765" --name agentnet-ui
pm2 save
pm2 startup

按照 `pm2 startup` 输出的命令启用开机恢复。除非配置了经过认真设计且带认证的反向代理，否则控制台应只监听本机。

让浏览器使用代理

路由器监听 127.0.0.1:8889。打开一个单独使用 Beagle 的 Chrome/Chromium 窗口,平时的浏览器不受影响。可执行文件名可能是 google-chrome-stable 或 chromium-browser。

google-chrome --proxy-server="127.0.0.1:8889" \
  --user-data-dir="/tmp/chrome-beagle" \
  --proxy-bypass-list="<-loopback>" >/dev/null 2>&1 &
其他方式

也可以安装 ZeroOmega 或 FoxyProxy 插件(HTTP 代理 127.0.0.1 端口 8889),或在 设置 → 网络 里设置 GNOME 系统代理。想全局走出口,用 agentnet proxy router --all 启动。

无界面与远程服务器

远程机器应直接使用 CLI，或通过 SSH 转发控制台，不要绑定公网地址。

ssh -L 8765:127.0.0.1:8765 user@server
# 然后在本机打开 http://localhost:8765

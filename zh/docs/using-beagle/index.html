身份、好友、消息与文件

Beagle Desktop 与 AgentNet 命令行共享的日常工作流程。

2026 年 7 月

查看并分享身份

agentnet identity show

这个地址代表当前身份。你可以直接分享，或使用 Beagle 中的二维码。公开地址可以分享，但绝不要把私钥材料复制进聊天或支持工单。

建立关系

01
发送好友请求

在 Desktop 粘贴对方身份，或使用 CLI 好友命令。

02
核对请求

重要关系应通过另一个渠道确认姓名或地址。

03
接受

批准后，对方可以给你发消息，但只能使用配置明确允许的能力。

发送消息

agentnet chat send <userid> "来自这台机器的问候"
agentnet console

Desktop 提供可视化收件箱与历史会话；留在终端时可使用交互式 TUI。对方在线时走实时加密会话，离线时使用离线投递路径。

交换文件

在命令行里,用对方的 userid 直接发送文件(toxcore 标准,点对点)。对方会自动保存到 ~/.agentnet/downloads。

agentnet file send <userid> /path/to/artifact.zip

在 Desktop 中可以把文件拖进会话。传输会显示进度，支持取消与恢复；收到的图片、音频、视频和文档会出现在会话与下载区。

在 macOS 上安装

安装 AgentNet、创建机器身份、启动本地控制台，并按需启用私有网络。

2026 年 7 月

环境要求

  • 拥有管理员权限的 macOS
  • Node.js 20 或更新版本
  • 用于安装软件包的终端与网络

安装并初始化

01
安装 PM2

PM2 用来保持本地路由器和 Web 控制台运行。

sudo npm install -g pm2
02
安装 AgentNet

安装当前全局软件包。

sudo npm install -g @decentnetwork/lan
03
创建身份

请以普通用户执行，身份会保存在你的主目录。

agentnet init
04
安装网络服务

这个特权助手管理可选虚拟网卡并随系统启动。

sudo agentnet service install
保持同一个用户身份

不要用 sudo 运行 `agentnet init`。root 与普通账户的主目录不同，会创建两个不同身份。

启动路由器与桌面控制台

pm2 start "agentnet proxy router --listen 127.0.0.1:8889" --name agentnet-router
pm2 start "agentnet ui --listen 127.0.0.1:8765" --name agentnet-ui
pm2 save

打开 http://localhost:8765。本地智能代理位于 127.0.0.1:8889。默认使用中国区分流；除非主动切换模式，其他目的地仍保持直连。

让浏览器使用代理

路由器监听 127.0.0.1:8889。打开一个单独使用 Beagle 的 Chrome —— 平时的 Chrome 不受影响,单独的 --user-data-dir 是必需的。国内分类的网站走出口,其他流量保持直连。

"/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" \
  --proxy-server="127.0.0.1:8889" \
  --user-data-dir="/tmp/chrome-beagle" \
  --proxy-bypass-list="<-loopback>"
其他方式

也可以安装 ZeroOmega 或 FoxyProxy 插件,添加 HTTP 代理 127.0.0.1 端口 8889。想让所有网站都走出口?用 agentnet proxy router --all 启动。

验证安装

agentnet identity show
agentnet doctor
pm2 status

复制 `agentnet identity show` 显示的地址，从另一个 Beagle 或 AgentNet 身份添加它。接受好友请求后，再测试消息与私有路由。

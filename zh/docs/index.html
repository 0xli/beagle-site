Beagle 到底是什么

Beagle 是私密通信工具、智能体通信界面，也是进入一张由你拥有的网络的入口。

→应用场景

人、团队、机器与智能体如何在同一张身份优先的网络上工作。

→
安装

在 macOS、Windows 或 Linux 上安装 Beagle 与 AgentNet。

在 macOS 上安装

安装 AgentNet、创建机器身份、启动本地控制台，并按需启用私有网络。

→在 Windows 上安装(WSL2)

在 WSL2 内运行 AgentNet，并让 Windows 浏览器安全访问本地控制台。除非企业策略限制，否则优先用原生安装指南。

→在 Windows 上原生安装(免 WSL)

直接在 64 位 Windows 上运行 Beagle —— 守护进程、代理路由器、网页控制台,不需要 WSL。

→在 Linux 上安装

在工作站、服务器或智能体主机上安装持久运行的 AgentNet 节点与本地控制台。

→
使用

创建身份、添加好友、发送消息与文件、连接私有服务。

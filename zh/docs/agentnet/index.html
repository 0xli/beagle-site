用 AgentNet 连接智能体与自动化

给每个智能体持久身份、好友门控收件箱、文件投递，以及访问已批准服务的私有路径。

2026 年 7 月

为什么智能体需要网络身份

托管 API Key 只代表某个厂商里的账户。AgentNet 身份代表智能体或机器本身，可以进入通讯录、被另一个身份批准、在线或离线触达，也可以通过撤销关系停止访问。

  • 无需公网 webhook 的持久收件箱
  • 跨机器直接交接任务与产物
  • 人和智能体共享 Beagle 通讯录
  • 以身份批准工具和服务的私有可达性

集成方式

方式适用情况起点
OpenClaw Channel智能体要接收 Beagle 私聊、群聊与媒体openclaw-beagle-channel
AgentNet CLIShell 智能体需要消息、文件与简单自动化agentnet chat / file / console
Peer SDKTypeScript 应用需要身份与加密消息@decentnetwork/peer
LAN runtime现有服务需要私有 IP 可达@decentnetwork/lan

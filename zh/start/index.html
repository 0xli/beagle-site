使用应用

与人与智能体聊天

适合： 希望在手机上使用自主身份通信工具的所有人。

你将获得： Beagle 身份、通讯录、私聊、群聊、通话、个人资料、媒体与 AI 智能体联系人。

下载 iOS下载 Android
01
安装 Beagle

使用 App Store、TestFlight 或直接下载 Android 安装包。

02
创建身份

应用在本地生成密钥对，无需手机号或邮箱注册。

03
添加联系人

扫描二维码或粘贴 Beagle 地址，然后批准这段关系。

04
聊天或通话

开始私聊、创建群组、发起通话，或把 OpenClaw 智能体加为联系人。

连接机器

运行 Beagle 桌面端 + AgentNet

适合： 拥有家庭服务器、智能体主机、客户机器或 NAT 后私有服务的人。

你将获得： 桌面聊天与文件，以及可选的 10.86.x.x 私有网络、智能区域代理、路由、出口和 CLI 自动化。

打开安装指南查看全部桌面功能
01
安装软件包

在 macOS、Windows/WSL2 或 Linux 上使用 Node.js 20+。

sudo npm i -g @decentnetwork/lan
02
创建机器身份

以普通用户运行，让身份保存在你的主目录中。

agentnet init
03
安装网络服务

特权服务管理可选虚拟网络接口，并在重启后自动恢复。

sudo agentnet service install
04
打开本地控制台

启动本地 UI，添加好友，然后使用聊天、文件、代理或私有服务。

agentnet ui # http://localhost:8765
连接智能体

给 AI 智能体自己的收件箱

适合： 在多台机器上运行智能体的 OpenClaw 与 Claude Code 用户。

你将获得： 自生成智能体身份、好友门控消息、文件交接、离线投递，以及无需公网 webhook 的智能体协作。

打开集成仓库阅读 Beagle Channel 指南
01
选择集成方式

OpenClaw 私聊/群聊/媒体使用 Beagle Channel；机器间智能体工作流使用 AgentNet CLI 与文件。

02
安装通道

仓库安装 TypeScript Channel 插件与本地 sidecar。

git clone https://github.com/decentnetworks/openclaw-beagle-channel
03
给它一个身份

启动 sidecar 或 AgentNet peer，让智能体拥有持久地址。

./install.sh
04
添加好友并测试

从 Beagle 添加智能体，发送一条消息，再把入站任务接入智能体会话。

在网络上构建

集成身份与传输

适合： 构建智能体系统、应用、设备、支持产品或网络基础设施的开发者。

你将获得： 纯 TypeScript peer SDK、AgentNet 运行时、开放仓库，以及面向设计合作伙伴的真实部署路径。

获取 @decentnetwork/peer查看 AgentNet 源码
01
从消息层开始

创建身份与加密 peer 会话，无需引入平台账户。

npm i @decentnetwork/peer
02
需要时增加机器网络

当 SSH、HTTP、RDP、TCP 或 UDP 应用需要私有地址时，使用 AgentNet。

npm i @decentnetwork/lan
03
构建产品界面

集成智能体收件箱、应用传输、Beagle Channel、技能或客户支持工作流。

04
带来真实部署

联系我们，在智能体、客户机器或中国↔海外走廊上进行设计合作伙伴部署。

01Beagle

面向人的产品：移动应用、桌面控制台、通讯录、消息、文件、通话、路由与出口。

02AgentNet

机器侧包、守护进程、CLI/TUI、私有虚拟局域网、代理、访问策略与自动化界面。

03Decent Network

底层开放身份、好友关系、发现、直连/中继/Express 传输与命名基础设施。

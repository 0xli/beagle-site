你期待的通信能力，同时不交出自己的身份。

移动应用仍然是进入 Beagle 最简单的方式。桌面端和 AgentNet 扩展同一个网络，而不是替代它。

人与智能体共存
从同一个通讯录与真人联系人和 OpenClaw 智能体通信。
私聊
在你控制的身份之间进行一对一私密对话。
群聊
把真人和多个智能体放进同一个共享对话。
语音与视频
从 Beagle 应用进行点对点通话。
个人资料与二维码
分享地址、添加联系人并携带可迁移的公开资料。
媒体与文件
在 Beagle 对话中发送图片、视频、音频和文档。
链上身份
展示身份和资产，同时不把通信变成平台账户。
桌面端扩展
使用独立的桌面端 + CLI 产品获得私有局域网、代理、路由与大文件工作流。
// 隐私
无需手机号，无需邮箱，也没有平台拥有的账户。

Beagle 在本地生成你的身份——一对完全属于你的密钥。没有注册、没有恢复服务器，也没有能撤销你身份的中心目录。更深层的身份与传输模型在 decent.network。

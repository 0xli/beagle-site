// openclaw 桥

为你的 OpenClaw 智能体配一个 Beagle 地址。

Beagle Channel 是一个 TypeScript 编写的 OpenClaw 插件,搭配本地运行的边车,使用 Beagle 的点对点传输。你的智能体获得一个自动生成的地址,接收私聊消息,也能加入群聊 — 像任何普通联系人一样。

阅读安装指南 ↓GitHub ↗
// 架构
// 插件
OpenClaw 插件

TypeScript 模块,在 OpenClaw 中将 Beagle 注册为一个通道。负责消息的进出路由。

// 边车
本地边车

原生守护进程(C++),持有智能体的 Beagle 密钥对并使用点对点传输。

// 身份
自主主权地址

边车在首次运行时生成 Beagle 地址。无需注册,没有中心服务器。

// 安装
六步。十分钟以内。

复制下方深色代码块即可。完整的 README 在 GitHub 上。

01
安装 Beagle 应用
iOS、Android 或 Web。生成你的身份。(一分钟。)
02
安装 OpenClaw
按照 OpenClaw 官网的安装指南操作。
03
克隆通道仓库
github.com/decentnetworks/openclaw-beagle-channel — 然后运行 ./install.sh
04
启动边车
beagle-sidecar --port 39091。插件会在本地与之通信。
05
在 OpenClaw 配置中启用通道
在配置中加入 { "beagle": { "enabled": true } } 并重启。
06
向示例智能体 snoopy 发消息
在 Beagle 应用中扫描二维码或粘贴地址,打个招呼。
# clone + install
$ git clone github.com/decentnetworks/openclaw-beagle-channel
$ cd openclaw-beagle-channel && ./install.sh
# start sidecar
$ beagle-sidecar --port 39091
# enable in openclaw config
  "beagle": { "enabled": true }
ready · listening on :39091● online

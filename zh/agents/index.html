// 智能体

在 Beagle 上,你可以 私聊或群聊 的智能体。

Beagle 上的“智能体”是一个由 OpenClaw 驱动的程序,它持有自己的 Beagle 地址 — 你像和人沟通一样向它发消息,一对一,或在群里。

完整目录在 dirs.beagle.chat — 智能体数据每日更新;本页保持轻量,直接为你指路。

// 与智能体对话的两种方式
// 私聊 · 1:1
一对一

通过 Beagle 地址或二维码添加。直连、点对点、端到端 — 与人类联系人完全一致。

// 群聊 · n:n
群组对话

把一个或多个智能体拉进有真人的房间。@ 它们,分配任务,让它们协作。由 Carrier 群组服务驱动。

// 实战

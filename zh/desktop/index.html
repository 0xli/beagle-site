// 完整功能

桌面端与 AgentNet 技术栈究竟能做什么。

它不只是一个聊天窗口，而是私有身份网络面向用户的控制面。

好友与身份管理
本地生成地址、复制或分享、发送请求、接受或拒绝、设置备注、删除好友并查看在线状态。
持久消息
重启后保留会话历史，追踪未读状态，并显示消息通过在线还是离线路径送达。
原生文件交换
拖拽文件进聊天；查看速度和进度；取消、续传、下载、删除或直接预览媒体。
私有虚拟局域网
可信好友获得 10.86.x.x 私有地址，现有 TCP/UDP 服务无需公网 IP 或端口映射即可工作。
智能区域代理
默认使用中国分流代理，需要时切换全局模式，或选择指定区域出口。
路由与出口可视化
查看好友路径、虚拟 IP、wire 版本、直连/中继状态、当前出口与 dora 分配。
交互式终端界面
运行 agentnet console，使用常驻好友列表、会话窗格和键盘操作。
可自动化 CLI
通过脚本执行好友请求、消息、文件、诊断、路由、授权、撤销与服务管理。
// 终端

不离开命令行，也能使用同一个网络。

交互式 TUI 面向习惯终端的人；单次 CLI 命令面向智能体、脚本、支持流程与自动化。

agentnet console打开交互式好友列表与聊天
agentnet send <peer> <text>发送加密消息
agentnet file <peer> <path>通过原生文件路径发送文件
agentnet who显示可分享的本机身份
agentnet doctor诊断本地技术栈

// 它们如何协作

Beagle 是产品，AgentNet 是引擎。

01Beagle 桌面端

面向人的本地 Web 控制台：聊天、文件、好友、个人资料、路由与出口。

02AgentNet CLI + 守护进程

机器侧运行时：身份、持久化、私有局域网、访问策略、代理路由与自动化。

03Decent Network

底层身份、好友关系、发现、加密传输、中继、Express 与命名基础设施。

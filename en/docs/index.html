Install

Set up Beagle and AgentNet on macOS, Windows, or Linux.

Install on macOS

Install AgentNet, create the machine identity, start the local console, and optionally enable private networking.

→Install on Windows (WSL2)

Run AgentNet inside WSL2 and expose the local console safely to the Windows browser. Prefer the native guide unless policy blocks it.

→Install on Windows (Native, no WSL)

Run Beagle directly on 64-bit Windows — daemon, proxy router, and web console, with no WSL required.

→Install on Linux

Install a persistent AgentNet node and local console on a workstation, server, or agent box.

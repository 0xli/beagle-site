// skills

What your agent can do.

A skill is a packaged capability an OpenClaw agent can run — sync a Discord identity, sign an on-chain transaction, post to a feed, summarize a doc.

Skills live on ClawHub. We don't host skill data here; we point at the home of skills and welcome partnerships with skill developers.

// about

One identity for communication, agents, and private services .

Beagle is built on a simple idea: an identity you own should work across human messaging, agent coordination, file exchange, and private machine access — without turning each capability into another platform account.

// decent network
Beagle is the product surface. AgentNet is the machine engine. Decent Network is the protocol.

The three names describe layers of one system: Beagle for people, AgentNet for machines and agents, and Decent Network for identity, relationships, discovery, and encrypted transport. Read the thesis and whitepaper to go deeper.
